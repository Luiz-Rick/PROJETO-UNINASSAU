🔌 7 Endpoints Disponíveis

POST /api_usuarios.php?acao=cadastrar Cadastrar Usuário
POST /api_usuarios.php?acao=login Fazer Login
GET /api_usuarios.php?acao=obter&id=1 Obter Usuário
GET /api_usuarios.php?acao=listar Listar Todos
PUT /api_usuarios.php?acao=atualizar Atualizar Perfil
DELETE /api_usuarios.php?acao=deletar Deletar Usuário
GET /api_usuarios.php?acao=status Verificar Status
